To get instant access to the private assignment vault, complete these three simple steps:

**Step 2: Authenticate**
Click the secure link below to verify your star. It will ask you to log in with your GitHub account so the system can confirm you've starred the repo.

**Step 3: Accept your Invite**

###  Troubleshooting

* **"I logged in but don't see the code!"** - You need to accept the collaboration invite first. Go to your [GitHub Notifications](https://github.com/notifications) to accept it.
