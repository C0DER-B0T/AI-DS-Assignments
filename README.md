### How to Unlock the Code

To get instant access to the private assignment vault, complete these three simple steps:

**Step 1: Drop a Star**
Scroll to the top right of this page and click the **Star** (⭐) button on this repository.

**Step 2: Authenticate**
Click the secure link below to verify your star. It will ask you to log in with your GitHub account so the system can confirm you've starred the repo.
 **[Unlock the Assignments Here](https://assignmentstar.streamlit.app)**

**Step 3: Accept your Invite**
Once verified, the system will automatically invite you to the private `AI-DS-Code-Vault` repository as a collaborator. Check your email or your GitHub notifications to accept the invite!

###  Troubleshooting
* **"It says I didn't star it!"** - Make sure you actually clicked the Star button on *this* specific public repository before clicking the login link.
* **"I logged in but don't see the code!"** - You need to accept the collaboration invite first. Go to your [GitHub Notifications](https://github.com/notifications) to accept it.
